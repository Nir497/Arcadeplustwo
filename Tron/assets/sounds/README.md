# TRON Sound Assets

These `.wav` files are placeholder-retro effects for the TRON Light Cycles game.
They are stored as uncompressed mono PCM so they can be loaded directly in the browser
without additional tooling.

## Suggested usage

- `menu-select.wav`: button clicks, setup selections
- `countdown-beep.wav`: `3`, `2`, `1` countdown ticks
- `round-start.wav`: `GO!` / round start cue
- `engine-loop.wav`: low-volume loop during active play
- `turn-blip.wav`: optional turn confirmation effect
- `crash.wav`: wall or trail collision
- `round-win.wav`: round winner stinger
- `round-draw.wav`: draw resolution cue

## Notes

- Files were synthesized locally for this project.
- Keep these filenames stable so the later sound implementation can reference them directly.
