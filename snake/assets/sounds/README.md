# Snake Sound Library

Generated retro-style mono WAV effects for the game.

Files:
- `start.wav`: game start / unpause cue
- `eat.wav`: food pickup
- `turn.wav`: direction change
- `pause.wav`: pause cue
- `resume.wav`: resume cue
- `game_over.wav`: loss sting
- `high_score.wav`: new high score celebration
- `menu_move.wav`: menu navigation / hover
- `confirm.wav`: confirm / button press

Reference metadata is also available in `manifest.json`.
