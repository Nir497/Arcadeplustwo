# Sound Library

Retro-style synthesized WAV effects for the Space Invaders build.

## Files

- `player_shoot.wav`: player laser
- `alien_shoot.wav`: alien projectile fire
- `alien_hit.wav`: alien explosion
- `player_hit.wav`: player ship destroyed
- `shield_hit.wav`: shield damage tick
- `alien_step_1.wav`: alien march step 1
- `alien_step_2.wav`: alien march step 2
- `alien_step_3.wav`: alien march step 3
- `alien_step_4.wav`: alien march step 4
- `ufo_loop.wav`: repeating UFO pass tone
- `ufo_hit.wav`: UFO destroyed
- `countdown_beep.wav`: countdown tick
- `countdown_go.wav`: start cue
- `wave_start.wav`: wave intro sting
- `extra_life.wav`: extra life reward
- `pause_toggle.wav`: pause on/off click
- `game_over.wav`: game over sting

## Notes

- All files are mono WAV at `22050 Hz`.
- Timing metadata is listed in [manifest.json](/Users/nirchuk/Downloads/arcadeplustwo/Space_invaders/assets/sounds/manifest.json).
- The library is asset-only right now; it is ready to be wired into `game.js`.
