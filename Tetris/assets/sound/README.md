Retro arcade sound set for the Tetris game.

Files:
- `start.wav`: start-run stinger
- `move.wav`: horizontal move tick
- `rotate.wav`: rotation chirp
- `soft_drop.wav`: soft-drop pulse
- `lock.wav`: piece lock thump
- `line_clear.wav`: 1-3 line clear reward
- `tetris_clear.wav`: 4-line clear reward
- `level_up.wav`: level-up fanfare
- `pause.wav`: pause toggle cue
- `top_out.wav`: game-over / top-out cue

All files are mono `.wav` assets generated locally for immediate use in the browser.
